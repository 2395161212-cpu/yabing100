## 设计规范

### 显示效果
- 图标显示尺寸：约 25px * 24px（系统自动缩放）
- 文字大小：约 12号字体（系统控制）
- 文字颜色：未选中 #6F6F6F，选中 #1296DB

### 图标文件要求
- 格式：PNG（支持透明背景）
- 源文件尺寸：建议 81px * 81px（或 162px * 162px）
- 文件大小：每个图标不超过 40KB
- 颜色：建议使用单色或简单配色，避免过于复杂

## 注意事项

1. 微信小程序会自动将图标缩放到合适大小显示
2. 文字大小和图标大小由系统控制，无法通过配置精确指定
3. 建议图标设计简洁，线条清晰，在小尺寸下也能清晰识别
4. 如需区分选中/未选中状态，可以分别准备两套图标
